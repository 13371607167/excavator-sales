## 1. 产品概述

挖掘机销售信息展示平台是一个专为挖掘机及配件销售商设计的产品展示网站。平台支持PC端浏览和移动端管理，用户可以在移动端方便地添加、编辑产品信息，在PC端进行产品展示。

- 主要用途：展示挖掘机及配件的销售信息，包括价格、状态、联系方式
- 目标用户：挖掘机销售商、工程机械经销商、二手设备交易商
- 核心价值：提供简洁高效的产品信息管理方式，帮助商家快速发布和展示产品

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 管理员 | 后台预设账号 | 添加、编辑、删除产品，管理联系方式 |
| 访客 | 无需注册 | 浏览产品列表，查看产品详情，获取联系方式 |

### 2.2 功能模块

1. **首页**: 产品分类导航、热门产品推荐、搜索功能
2. **产品列表页**: 产品筛选、排序、列表展示
3. **产品详情页**: 产品图片、价格、状态、联系方式
4. **移动端管理页**: 添加产品、编辑产品、删除产品、管理联系方式

| 页面名称 | 模块名称 | 功能描述 |

## 3. 核心流程

### 3.1 访客浏览流程

用户访问网站 → 浏览产品列表/搜索产品 → 点击查看详情 → 查看价格和联系方式 → 联系商家

### 3.2 管理员发布流程

管理员登录移动端 → 填写产品信息 → 上传产品图片 → 设置价格/状态 → 发布产品

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**: 工程黄 (#F5A623) 作为品牌色，深灰 (#2C3E50) 作为主文字色，浅灰 (#ECF0F1) 作为背景色
- **辅助色**: 橙色 (#E67E22) 用于强调，绿色 (#27AE60) 表示在售，红色 (#E74C3C) 表示已售
- **按钮风格**: 圆角矩形按钮，带轻微阴影，hover时有上浮效果
- **字体**: 标题使用 Noto Sans SC Bold，正文使用 Noto Sans SC Regular
- **布局风格**: 卡片式布局，顶部导航，响应式网格
- **图标风格**: 线性图标，与工程机械设备风格统一

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 首页 | Hero区域 | 全屏背景图，大标题居中，渐变遮罩 |
| 首页 | 产品分类 | 两个大卡片，图标+文字，hover放大效果 |
| 首页 | 热门产品 | 4列网格卡片，图片+标题+价格+状态标签 |
| 产品列表页 | 筛选栏 | 下拉选择器，价格区间输入，状态筛选按钮组 |
| 产品列表页 | 产品列表 | 响应式网格，卡片带阴影和圆角 |
| 产品详情页 | 图片展示 | 大图轮播，缩略图导航 |
| 产品详情页 | 产品信息 | 左侧图片，右侧信息，底部联系方式卡片 |
| 移动端管理页 | 产品表单 | 单列布局，输入框带图标，提交按钮固定底部 |

### 4.3 响应式设计

- **设计原则**: 移动优先，桌面端增强
- **断点设置**: 
  - 移动端: < 768px (单列布局)
  - 平板: 768px - 1024px (双列布局)
  - 桌面: > 1024px (多列网格布局)
- **触控优化**: 移动端按钮最小高度44px，间距适中，支持滑动操作

### 4.4 产品状态标识

- **在售**: 绿色标签，显示价格
- **已售**: 红色标签，显示"已售"
- **面议**: 橙色标签，显示"价格面议"
